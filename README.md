# Ecoleta
An application to connect companies and people in favor of the environment. With that, we are contribuiting to point out the nearest drop-off locations.

The project is divided in 3 parts:
- Server
- Mobile app
  - Used to search drop-off locations.
- Web app
  - Used to create drop-off locations

## Technologies

### Server
Node.js (Typescript)

### Mobile app
React Native (Typescript)

### Web app
ReactJS (Typescript)
